# MooVit

A project for managing and tracking activities.

## Getting Started

### Prerequisites
- Node.js
- npm or yarn

### Installation
```bash
git clone https://github.com/ShubhangiRoy12/MooVit.git
cd MooVit
npm install
```

### Running
```bash
npm start
```
