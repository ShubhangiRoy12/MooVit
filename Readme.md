# MooVit 🚍

MooVit is a smart public transportation and route-planning application designed to help users navigate city transport systems efficiently. The project aims to simplify daily commuting by providing route information, transit details, and an improved travel experience.

## Features

* 🗺️ Route planning and navigation
* 🚏 Bus stop and transit information
* 📍 Location-based transport suggestions
* 🔎 Search for routes and destinations
* 📱 Responsive and user-friendly interface
* ⚡ Fast and lightweight application

## Installation

Clone the repository:

```bash
git clone https://github.com/ShubhangiRoy12/MooVit.git
```

Navigate to the project folder:

```bash
cd MooVit
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

## Usage

1. Open the application in your browser.
2. Enter your starting location and destination.
3. View suggested routes and transport options.
4. Choose the best route for your journey.
